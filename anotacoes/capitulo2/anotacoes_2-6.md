# Por que Funções são Úteis, e como Usá-las Efetivamente

## Por que usar Funções?

Usar funções oferece um número de benefícios que as fazem úteis em programas complexos:

- Funções reduzem um programa complicado em pedaços menores e mais fáceis de se gerenciar
- Funções permitem a reutilização de código e a redução de código escrito, além de permitir compartilhamento de código entre programas
- Funções ajudam para testes menores e mais eficientes
- Funções permitem a extensão de programas para caso além do plano inicial
- Funções são mais fáceis de serem entendidas para serem utilizadas

## Guias para Escrever Funções

- Um grupo de comandos que aparecem mais de uma vez no programa devem ser transformados em uma função
    - Exemplo: leitura do usuário em repetidas vezes
- Um código que tem entradas e saídas bem definidas é um bom candidato para uma função
    - Exemplo: organização de uma lista de itens
- Uma função deve fazer uma tarefa somente
- Quando uma função se torna muito longa ou complicada, pode ser dividida em sub-funções
